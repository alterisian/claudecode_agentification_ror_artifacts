# Jumpstart

**Repo:** https://github.com/jumpstart-rails/jumpstart-rails
**Command:** `rails new myapp -m jumpstart/template.rb`

## Best for
SaaS products that need to ship fast. Opinionated but stays close to Rails conventions.

## What you get
- Authentication (Devise) with multiple OAuth providers out of the box
- Subscription billing via Stripe or Paddle
- Multi-tenancy / team accounts
- Tailwind CSS + Hotwire
- Admin panel
- Notifications (email, SMS, web push)
- Fly.io and Heroku deploy configs

## Trade-offs
- More magic than a greenfield app — harder to unpick later
- Paid pro version unlocks the full feature set
