# Suspenders

**Repo:** https://github.com/thoughtbot/suspenders
**Command:** `gem install suspenders && suspenders myapp`

## Best for
Teams that want thoughtbot's proven conventions without the SaaS extras.

## What you get
- RSpec + FactoryBot pre-configured
- Brakeman, Bundler Audit security checks
- Rubocop with thoughtbot style guide
- PostgreSQL by default
- Heroku-ready Procfile and config

## Trade-offs
- Heroku-centric — some rework needed for Fly.io
- Minimal UI layer — no auth or billing, just the skeleton
- Reflects thoughtbot's choices, which may not match yours
